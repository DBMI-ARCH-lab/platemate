platemate
=========

PlateMate: crowdsourcing nutritional analyses from food photographs

## Setup

### Install requirements
    sudo apt-get install libjpeg-dev
    pip install -r requirements.txt

This should install:
    * Django 1.2.5
    * Pillow (latest)
    * Boto (latest)
    * httpagentparser (latest)
    * oauth latest
