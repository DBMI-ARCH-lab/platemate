Before running, make sure you do the following:

* Install Django 1.2.5 http://www.djangoproject.com/download/
* Install http://code.google.com/p/boto/
* Install http://pypi.python.org/pypi/httpagentparser
* Install http://pypi.python.org/pypi/oauth/1.0.1
* Modify local_settings.example.py to your paths and rename to local_settings.py
* From the platemate directory, run "python manage.py syncdb --noinput
